# Changelog

所有重要的项目更改都会记录在此文件中。

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
项目版本遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-15

### 新增

- 实现核心贝叶斯同化算法
- 添加 3D-VAR 三维变分同化算法
- 添加 4D-VAR 四维变分同化算法
- 添加 EnKF 集合卡尔曼滤波算法
- 添加混合同化算法（自适应组合多种算法）
- 实现并行计算框架
  - 区域分解并行（Block Parallel）
  - Dask 分布式计算
  - MPI 多机并行
  - Ray 分布式计算
- 添加数据适配器模块
  - WRF 数据适配器
  - 观测数据适配器
  - 网格数据适配器
- 实现质量控制模块
  - 数据验证器
  - 异常检测
- 实现风险评估模块
  - 风险评估器
  - 风险热力图生成
- 添加时间序列分析模块
  - 趋势分析
  - 异常检测
  - 预测功能
- 实现可视化模块
  - 静态图表
  - 动画演示
  - 仪表盘
- 添加 API 接口
  - CLI 命令行接口
  - REST API（FastAPI）
  - Web 界面（Flask）
- 实现工作流管理
  - 批处理工作流
  - 流水线工作流
  - 流式处理工作流
- 添加数据源模块
  - 卫星数据源
  - 雷达数据源
  - 地面站数据源
- 添加 GPU 加速支持
  - CUDA 加速
  - JAX 加速
- 实现容器化部署
  - Dockerfile
  - Docker Compose
  - 多环境支持

### 更改

- 重构项目结构，遵循单一职责原则
- 优化核心算法性能
- 改进内存管理

### 修复

- 修复导入错误
- 修复类型错误
- 修复维度不匹配问题
- 修复并行计算中的死锁问题

### 移除

- 移除 deprecated 的 API
- 移除未使用的依赖
