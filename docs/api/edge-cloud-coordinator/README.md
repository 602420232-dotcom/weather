# edge-cloud-coordinator

本目录包含边云协调器（Edge-Cloud Coordinator）的 API 文档。边云协调器是 UAV 路径规划系统中的核心调度组件，负责协调边缘端（UAV 机载计算单元）与云端（中心服务器）之间的任务分配、数据同步和资源调度，确保在带宽受限和延迟不稳定的环境下实现高效的计算卸载与协同工作。

| 文件 | 描述 |
|------|------|
| [coordinator.md](coordinator.md) | 边云协调器 API 接口文档，包含任务分发、资源调度、边缘节点注册、数据同步策略等接口定义 |
