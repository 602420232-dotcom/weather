# meteor-forecast-service

本目录包含气象预报服务（Meteorological Forecast Service）的 API 文档。气象预报服务基于 WRF 数值预报模型和数据同化结果，生成指定区域、指定时效的高分辨率气象预报产品，为 UAV 飞行安全和路径规划提供风场、温度、湿度、降水、能见度等关键气象要素预测。

| 文件 | 描述 |
|------|------|
| [forecast.md](forecast.md) | 气象预报服务 API 接口文档，包含预报任务提交、预报结果查询、区域天气预报获取、集合预报等接口定义 |
