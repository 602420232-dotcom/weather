# path-planning-service

本目录包含路径规划服务（Path Planning Service）的 API 文档。路径规划服务是 UAV 飞行任务的核心决策模块，结合实时气象数据、地形信息、禁飞区约束以及 UAV 性能参数，为无人机生成安全、高效的三维飞行路径，支持动态避障和路径重规划。

| 文件 | 描述 |
|------|------|
| [path.md](path.md) | 路径规划服务 API 接口文档，包含路径生成请求、路径优化、动态重规划、路径评估等接口定义 |
