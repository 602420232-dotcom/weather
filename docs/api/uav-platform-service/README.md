# uav-platform-service

本目录包含 UAV 平台服务（UAV Platform Service）的 API 文档。UAV 平台服务是系统的业务核心，负责管理无人机资产、飞行任务、用户权限、数据来源和历史记录等关键业务实体，为上层路径规划和气象服务提供基础数据和业务支撑。

| 文件 | 描述 |
|------|------|
| [auth.md](auth.md) | 认证与授权 API 接口文档，包含用户登录、Token 管理、权限校验等接口定义 |
| [data-sources.md](data-sources.md) | 数据源管理 API 接口文档，包含气象数据源注册、配置、状态监控等接口定义 |
| [drone.md](drone.md) | 无人机管理 API 接口文档，包含无人机注册、状态监控、参数配置、机队管理等接口定义 |
| [history.md](history.md) | 历史记录 API 接口文档，包含飞行记录、任务历史、操作日志等查询接口定义 |
| [path.md](path.md) | 路径管理 API 接口文档，包含路径存储、路径模板、路径版本管理等接口定义 |
| [task.md](task.md) | 任务管理 API 接口文档，包含飞行任务创建、派发、执行跟踪、任务状态管理等接口定义 |
| [user.md](user.md) | 用户管理 API 接口文档，包含用户注册、信息管理、角色分配等接口定义 |
