# uav-weather-collector

本目录包含 UAV 气象采集服务（UAV Weather Collector）的 API 文档。UAV 气象采集服务运行在无人机机载端，负责通过搭载的气象传感器实时采集温度、湿度、气压、风速、风向等大气参数，并将采集数据回传至地面站或云端，作为数据同化和气象预报的重要观测输入。

| 文件 | 描述 |
|------|------|
| [weather.md](weather.md) | UAV 气象采集服务 API 接口文档，包含传感器数据采集、数据封装与传输、采集任务控制、设备校准等接口定义 |
