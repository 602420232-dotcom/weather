# wrf-processor-service

本目录包含 WRF 处理服务（WRF Processor Service）的 API 文档。WRF（Weather Research and Forecasting）处理服务负责驱动和管理 WRF 数值天气预报模型的运行，包括模型参数配置、初始场和边界条件准备、模式积分计算、后处理与可视化输出等全流程管理。

|------|------|
| [wrf.md](wrf.md) | WRF 处理服务 API 接口文档，包含 WRF 模型配置、模拟任务提交、运行状态监控、输出结果获取等接口定义 |
