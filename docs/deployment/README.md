# deployment

| 文件 | 描述 |
|------|------|
| [DEPLOYMENT.md](DEPLOYMENT.md) | 系统部署主文档，涵盖 Docker 容器化部署、服务编排、环境变量配置等完整部署流程 |
| [DEPLOY_GUIDE.md](DEPLOY_GUIDE.md) | 部署操作指南，提供从源码构建到线上发布的分步操作说明和最佳实践 |
| [DISASTER_RECOVERY_PLAN.md](DISASTER_RECOVERY_PLAN.md) | 灾难恢复计划，描述系统故障分级、数据备份策略、服务恢复流程和应急响应方案 |
