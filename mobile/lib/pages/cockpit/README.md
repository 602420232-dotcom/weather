# pages/cockpit

驾驶舱页面，提供实时气象数据可视化，包括风速风向动态仪表、二维风羽图、飞线轨迹和深度剖面对比图表。

## 关键文件

| 文件 | 说明 |
|------|------|
| `cockpit_page.dart` | 驾驶舱 `CockpitPage`，使用 `fl_chart` 和 `flutter_map` 展示实时气象仪表 |

### 可视化组件

| 组件 | 技术 | 说明 |
|------|------|------|
| 风速风向仪表 | `fl_chart` 雷达图 | 模拟实时动态更新（3s 间隔） |
| 二维风羽图 | `flutter_map` + `LatLng` | 地图叠加气象图层 |
| 飞线轨迹 | 自定义绘制 | 无人机飞行路径展示 |
| 深度剖面 | `fl_chart` 折线图 | 风速/温度/湿度随高度变化 |
