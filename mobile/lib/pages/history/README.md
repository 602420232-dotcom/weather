# pages/history

历史记录页面，展示已执行任务的历史清单，支持搜索、状态筛选和日期范围过滤。

## 关键文件

| 文件 | 说明 |
|------|------|
| `history_page.dart` | 历史记录页 `HistoryPage`，支持搜索、状态和日期范围三重过滤 |

### 功能

| 功能 | 说明 |
|------|------|
| 搜索 | 按任务名称/编号模糊搜索 |
| 状态筛选 | 全部、成功、失败、进行中 |
| 日期筛选 | 开始日期 ~ 结束日期范围选择 |
| 记录详情 | 任务编号、名称、开始/结束时间、状态标签、飞行距离、执行耗时 |
