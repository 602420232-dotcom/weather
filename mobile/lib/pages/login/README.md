# pages/login

登录页面，提供用户名/密码表单认证，支持密码显示切换和错误提示。

## 关键文件

| 文件 | 说明 |
|------|------|
| `login_page.dart` | 登录页 `LoginPage`，基于 Riverpod 调用 `AuthService.login()` |

### 功能

| 功能 | 说明 |
|------|------|
| 表单验证 | 用户名和密码非空校验 |
| 密码显示切换 | 眼睛图标切换密码明文/密文 |
| 登录流程 | 调用认证服务，成功后写入 Token 并跳转到首页 |
| 错误处理 | 显示服务端返回的错误消息 |
| 默认凭据 | 预填 `admin / admin123` |
