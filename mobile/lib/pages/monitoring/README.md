# pages/monitoring

系统监控页面，展示服务健康状态、资源使用率和算法性能指标。

## 关键文件

| 文件 | 说明 |
|------|------|
| `monitoring_page.dart` | 系统监控页 `MonitoringPage`，使用 `fl_chart` 展示实时系统指标 |

### 功能

| 功能 | 说明 |
|------|------|
| 服务状态 | 在线/离线服务数量、健康评分 |
| 资源监控 | CPU、内存、磁盘使用率（模拟 5s 刷新） |
| 活跃任务 | 当前活跃任务数 |
| 算法性能 | 柱状图展示各算法（3D-VAR / EnKF / VRPTW / A* / DWA 等）执行时间和成功率 |
