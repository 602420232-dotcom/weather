# pages/weather

气象数据页面，提供基于地图的气象可视化、高空多层数据展示和预警信息。

## 关键文件

| 文件 | 说明 |
|------|------|
| `weather_page.dart` | 气象数据页 `WeatherPage`，支持时间/高度层级切换的地图气象可视化 |

### 功能

| 功能 | 说明 |
|------|------|
| 地图气象图层 | 基于 `flutter_map` 叠加风速风向可视化 |
| 时间轴选择 | 切换不同预报时间点的气象数据 |
| 高度层级 | 100m / 500m / 1000m / 1500m 多高度层数据 |
| 实时指标 | 风速、风向、温度、湿度、湍流度、能见度 |
| 趋势图表 | `fl_chart` 折线图展示风速/温度随时间变化 |
| 预警列表 | 气象预警信息列表展示 |
