# pages/data_sources

数据源管理页面，展示和管理气象数据来源（地面站、浮标、卫星、雷达等），支持分类 Tab 切换。

## 关键文件

| 文件 | 说明 |
|------|------|
| `data_source_page.dart` | 数据源页 `DataSourcePage`，使用 `TabBar` 分类展示不同来源类型 |

### 数据源类型

| Tab | 类型 | 说明 |
|-----|------|------|
| 全部 | — | 所有数据源 |
| 地面站 | `ground_station` | 地面气象观测站 |
| 浮标 | `buoy` | 海上浮标气象数据 |
| 卫星 | `satellite` | 卫星遥感数据 |
| 雷达 | `radar` | 气象雷达数据 |
