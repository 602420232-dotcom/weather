# pages/drones

无人机管理页面，展示无人机列表及其状态信息（在线/执行任务/待命/维护），支持筛选和详情查看。

## 关键文件

| 文件 | 说明 |
|------|------|
| `drones_page.dart` | 无人机管理页 `DronesPage`，支持状态筛选（在线/待命/维护） |

### 功能

| 功能 | 说明 |
|------|------|
| 状态筛选 | 按在线、待命、维护状态过滤无人机列表 |
| 卡片展示 | 电池电量、型号、最大载荷、续航、速度等信息 |
| 模拟数据 | 内置猎鹰/天鹰/闪电/海燕系列共 6 架无人机 |
