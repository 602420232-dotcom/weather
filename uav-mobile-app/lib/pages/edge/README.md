# pages/edge

边云协同管理页面，管理边缘计算节点的连接状态、任务队列和气象风险评估。

## 关键文件

| 文件 | 说明 |
|------|------|
| `edge_sync_page.dart` | 边云协同页 `EdgeSyncPage`，调用 `EdgeCoordinatorService` 管理边缘节点 |

### 功能

| 功能 | 说明 |
|------|------|
| 连接状态 | 实时显示云端/边缘端连接状态（在线/离线） |
| 统计面板 | 任务队列大小、已完成数、缓冲区使用率 |
| 气象风险评估 | 提交当前气象数据到边缘节点，返回风险等级和警告信息 |
| 状态刷新 | 手动刷新获取最新边缘节点系统状态 |

### 风险评估等级

| 等级 | 颜色 | 说明 |
|------|------|------|
| `low` | 绿色 | 适合飞行 |
| `medium` | 黄色 | 注意飞行安全 |
| `high` | 橙色 | 不建议飞行 |
| `severe` | 红色 | 禁止飞行 |
