# pages/home

首页模块，作为应用的主入口仪表盘。

## 关键文件

| 文件 | 说明 |
|------|------|
| `home_page.dart` | 首页 `HomePage`，展示系统概览、快速操作入口、气象预警、任务统计、核心能力列表 |

### 页面结构

| 区域 | 说明 |
|------|------|
| 系统概览 | 4 格卡片：在线无人机数、待执行任务数、系统状态、今日完成数 |
| 快速操作 | 4 个快捷入口：路径规划、气象数据、任务管理、无人机管理 |
| 气象预警 | 实时风速、风险等级、更新时间 |
| 任务统计 | 今日完成、本周完成、成功率 |
| 核心能力 | WRF 气象预报、AI 气象订正、三层路径规划、数据同化、边云协同、实时监控 |
