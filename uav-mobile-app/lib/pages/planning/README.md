# pages/planning

路径规划页面，提供地图可视化、航点编辑、多算法路径规划和气象风险评估功能。

## 关键文件

| 文件 | 说明 |
|------|------|
| `planning_page.dart` | 路径规划页 `PlanningPage`，基于 `flutter_map` 实现地图交互 |

### 功能

| 功能 | 说明 |
|------|------|
| 地图交互 | 基于 OpenStreetMap 瓦片地图，支持添加/移动航点 |
| 多算法规划 | VRPTW（全局任务分配）、A*（全局路径）、DWA（局部避障）、全链路 |
| 气象数据面板 | 实时显示风速、温度、湿度、风险等级 |
| 规划结果展示 | 路径可视化、总距离、预计时间 |
