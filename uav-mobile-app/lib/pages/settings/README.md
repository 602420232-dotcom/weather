# pages/settings

设置页面，提供应用配置管理，包括服务器地址、主题切换、离线功能开关等。

## 关键文件

| 文件 | 说明 |
|------|------|
| `settings_page.dart` | 设置页 `SettingsPage`，分组展示各项配置 |

### 配置分组

| 分组 | 配置项 |
|------|--------|
| 服务器配置 | API 服务器地址、边缘计算节点地址 |
| 系统 | 离线地图缓存开关、地图类型选择、暗黑模式开关 |
| 数据同步 | 自动同步开关 |
