# pages/tasks

任务管理页面，展示任务列表，支持按状态筛选、新建任务。

## 关键文件

| 文件 | 说明 |
|------|------|
| `tasks_page.dart` | 任务管理页 `TasksPage`，支持状态筛选和新增任务弹窗 |

### 功能

| 功能 | 说明 |
|------|------|
| 状态筛选 | 按全部、待分配、已分配、执行中、已完成过滤 |
| 任务卡片 | 类型标签（物流/巡检/救援/监测）、优先级、航点数 |
| 新建任务 | 弹窗表单创建新任务 |
