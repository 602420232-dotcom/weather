# UAV Path Planning System - 无人机路径规划系?

##  项目概述

基于 WRF 气象驱动的无人机 VRP车辆路径问题智能路径规划系统?

**核心特?*:
- ?**气象驱动**: 集成 WRF 气象模型实时获取风场温度降水等数据
- ?**智能规划**: 融合 VRPTWDE-RRT*DWA 等算?
-  **数据同化**: 贝叶斯数据同化技术融合多源气象观测
-  **实时监控**: 可视化监控无人机状态和任务进度
-  **联邦学习**: Edge-Cloud 协同支持端侧离线路径规?

**技术栈**:
- **前端**: Vue 3 + Vite + Element Plus
- **后端**: Spring Boot 3.2 + Java 17
- **数据?*: MySQL 8.0 + Redis
- **算法**: Python 3.8+ (NumPy, SciPy, TensorFlow)
- **容器**: Docker + Kubernetes
